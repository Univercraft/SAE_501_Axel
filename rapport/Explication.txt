Migration du projet vers Vite + Tailwind CSS 4

1. Pourquoi utiliser Vite ?
Vite est un outil moderne de développement web qui permet de créer des applications rapidement, avec un rechargement instantané et une configuration minimale. Il facilite l’organisation du code en composants réutilisables et accélère le développement.

2. Pourquoi utiliser Tailwind CSS 4 ?
Tailwind CSS 4 permet de styliser rapidement les pages grâce à des classes utilitaires directement dans le HTML ou JSX. Il n’est plus nécessaire de créer un fichier de configuration par défaut, ce qui simplifie la prise en main. Tailwind facilite la création d’interfaces responsives et modernes.

3. Migration d’une page HTML vers React + Tailwind
Chaque page HTML (ex : page_17.html) est transformée en composant React. Les styles sont appliqués avec les classes Tailwind, ce qui rend le code plus lisible et plus facile à maintenir. Les images sont importées comme modules, ce qui évite les erreurs de chemin.

4. Navigation
La navigation entre les pages se fait avec React Router, ce qui permet une expérience utilisateur fluide sans rechargement complet de la page.

5. Avantages de ces choix
- Code plus propre et modulaire
- Développement plus rapide
- Facilité d’ajout de fonctionnalités interactives
- Meilleure maintenabilité

En résumé, ce choix technologique permet d’obtenir une application web moderne, rapide et évolutive, adaptée à un projet de livre animé en ligne.
